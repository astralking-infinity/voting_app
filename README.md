# Voting App
Django-React powered voting app

### Requirements
- python3.7
- pipenv
- npm

### Usage
`$ git clone https://github.com/astralking-infinity/voting_app`

`$ cd voting_app`

`$ pipenv install`

`$ pipenv run python manage.py migrate`

`$ pipenv run python manage.py runserver`

`$ cd frontend`

`$ npm install`

`$ npm run start`
